# Roleta de Brindes do Rotaract

Uma webpage simples de React com Vite para atração e engajamento de novos associados para o Rotaract.

A roleta é bem simples contendo campos para que a pessoa informe seus dados de contato para que seja possível entrar em contato posteriormente e ver se a pessoa tem interesse em participar das atividades do Rotaract, e uma roleta que gira aleatoriamente com brindes cadastrados.
